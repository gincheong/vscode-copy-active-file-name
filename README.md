# VS Code Copy Active File Name (No Ext)

This extension allows you to copy the name of the active file to the clipboard without its extension.

## Features

- Copy current file name (excluding extension) to clipboard.

## Usage

1. Open a file in VS Code.
2. Open Command Palette (`Cmd+Shift+P` or `Ctrl+Shift+P`).
3. Run `Copy Active File Name (No Ext)`.
